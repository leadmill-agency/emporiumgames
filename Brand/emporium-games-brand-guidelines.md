# Emporium Games Brand Guidelines

## Brand Core

**Brand idea:**  
Emporium Games should feel like a **real-world gaming hall**.  
Not a generic gamer site. Not a sleek startup. Not a fake ecommerce template.

The right vibe is:

- old-school fantasy
- neighborhood gaming hall
- slightly gritty late-night energy
- immersive, social, memorable

The storefront already points in this direction. The sign is bold and readable. The yellow-on-dark color treatment is loud in a good way. The window treatment feels medieval / gothic / tavern-like. The website should lean into that instead of fighting it.

---

Emporium Games is a physical place where people come to play, compete, hang out, and host events.

## Web Design Principles — Avoiding the AI / Template Look

The website must feel **owner-built and handcrafted**, not auto-generated. The following principles are drawn from studying award-winning dark venue sites (Dante NYC, The Dead Rabbit, Smuggler's Cove) and the patterns that make AI-generated or template-driven sites immediately recognizable.

### The Problem with AI-Generated Sites

AI tools produce sites that are technically correct but visually generic. They default to:

- Perfectly uniform spacing on every section
- Repetitive layout patterns (centered heading → subtitle → 3-column grid, repeat)
- Emoji icons as visual shorthand
- Identical card components everywhere
- Symmetrical grids with no visual tension
- Every section padded the same amount
- Decorative elements applied mechanically (gold line under every heading)
- Too many sections crammed onto one page
- Hover effects on everything
- Cookie-cutter navigation (logo left, links center, CTA right)

These patterns are instantly recognizable because they lack **intentional irregularity** — the small imperfections and asymmetries that signal human decision-making.

---

### Layout Principles

**Break the grid monotony.** Not every section should be a centered heading followed by evenly spaced cards. Vary the layout structure from section to section:

- One section might be a full-bleed photograph with text overlaid
- The next might be a tight asymmetric two-column (60/40 or 70/30) with lots of whitespace
- Another might be a single large statement with no supporting elements
- Use offset positioning — an image that bleeds past its container, text that sits off-center

**Use asymmetry intentionally.** Symmetry reads as mechanical. Real designers create visual tension:

- Offset images that don't center-align with their text
- Unequal column widths in split layouts
- Elements that break out of containers or overlap section boundaries
- Varied card sizes within the same grid (one large, two small)

**Vary section rhythm.** Not every section needs the same vertical padding. Some sections should breathe with massive whitespace. Others should feel tighter and more compressed. This creates pacing — like chapters in a book, not rows in a spreadsheet.

**Show restraint.** Fewer sections, each with more visual impact, beats cramming every possible block onto one page. Dante NYC and Dead Rabbit show only what matters and let each section command attention.

---

### Typography Principles

**Let type do the heavy lifting.** Instead of relying on icons or decorative elements, use dramatic scale contrasts in typography:

- Hero headlines should be oversized and cinematic
- Subtext should be noticeably smaller — not just one step down, but a dramatic shift
- Section headings don't all need to be the same size or treatment
- Some headings might be massive and left-aligned; others small and uppercase

**Mix typographic treatments.** Avoid applying the same heading style to every section:

- One section heading might be huge, light-weight, and lowercase
- Another might be small, bold, uppercase, and letter-spaced
- Use italics, different weights, or mixed case to create variety
- Pull quotes or featured text should feel distinct from standard headings

**Avoid uniform text blocks.** Real sites vary text width, alignment, and density:

- Some text blocks are narrow (max 500px) for readability
- Others span wider for impact
- Not everything needs to be centered — left-aligned text often feels more natural and editorial

---

### Visual Texture and Depth

**Layer backgrounds.** Flat, single-color section backgrounds look AI-generated. Instead:

- Use subtle CSS noise/grain overlays on dark backgrounds
- Layer multiple gradients (radial + linear) at low opacity
- Vary background darkness section to section — not just alternating between two shades
- Consider a subtle vignette effect (darker at edges)

**Create atmospheric depth.** The tavern should feel like it has *walls, not panels*:

- Use box-shadows and layered borders to create physical depth
- Background elements at different "distances" (parallax-like, even if subtle)
- Soft light sources implied through gradient placement
- Occasional texture references (wood grain, stone, parchment) through CSS patterns

**Use photography as atmosphere, not decoration.** When real photos are available:

- Full-bleed images that establish mood, not thumbnails in cards
- Overlaid text on photographs (with proper contrast)
- Images that bleed past containers or sit behind content
- Cropping that feels editorial, not stock-photo-perfect

---

### Icon and Decorative Element Rules

**No emoji icons.** Ever. They immediately signal AI-generated content.

**Preferred approaches (in order):**

1. **No icons at all** — let strong headings and copy carry the section. This is what most award-winning venue sites do.
2. **Simple SVG line icons** — thin, single-color, consistent stroke weight. Used sparingly.
3. **Custom illustrated elements** — if the brand develops shield/crest/banner motifs, use those as section markers.
4. **Typographic markers** — a gold diamond (◆), a thin rule, or a simple dash can work as minimal separators.

**Decorative restraint:**

- Don't put a gold underline after every heading — pick specific moments
- Section dividers should vary or be absent entirely between some sections
- Ornamental elements should feel earned, not automated
- If a decorative element appears on every section, it's too frequent

---

### Navigation Personality

The nav should have character, not just function:

- Consider a nav that reveals on scroll rather than being permanently fixed
- The logo treatment can be more than just text — consider a wordmark with character
- Navigation links don't have to be uniform uppercase — mixed case or sentence case can feel more human
- The "Book an Event" CTA in the nav should feel distinct, not just a gold rectangle
- Consider hiding the nav entirely on the hero and revealing it only when the user scrolls

---

### Animation and Motion

**Restraint over spectacle.** AI sites add hover effects to everything. Real designer sites are selective:

- Not every card needs a hover lift effect
- Scroll animations should be subtle and infrequent — fade-ins on key elements only, not on every single component
- Page transitions and section reveals should feel gentle, not performative
- Parallax: use at most on one element, or skip entirely
- If something animates, it should feel like it has weight and purpose

---

### Anti-Patterns Checklist

Before finalizing the website, check for these AI/template tells:

- [ ] Are all sections using the same layout pattern? → Vary them
- [ ] Are all cards identical in size and structure? → Mix sizes
- [ ] Are there emoji icons anywhere? → Remove them
- [ ] Does every heading have the same decorative underline? → Vary or remove
- [ ] Is every section centered? → Add left-aligned or asymmetric sections
- [ ] Does every section have the same padding? → Vary the rhythm
- [ ] Are there hover effects on every interactive element? → Be selective
- [ ] Does every element fade in on scroll? → Limit to key moments
- [ ] Is the nav a standard logo-links-CTA bar? → Add personality
- [ ] Are there more than 6-7 sections on the homepage? → Consider cutting
- [ ] Do all backgrounds look flat and uniform? → Add texture and depth
- [ ] Does the site look like it could be any business with a color swap? → Add specificity

---

### Reference Sites

These venues demonstrate the principles above. Study their layouts, not their content:

- **Dante NYC** (dante-nyc.com) — asymmetric layouts, custom fonts, editorial pacing, restraint
- **The Dead Rabbit** (thedeadrabbit.com) — irregular grids, cultural specificity, mixed-case type, layered depth
- **Smuggler's Cove** (smugglerscovesf.com) — narrative scroll, documentary photography, varied section structures, owner-driven voice

Key patterns from these sites:

- Photography-first design (images are atmospheric, not decorative)
- Narrative flow (the page tells a story, not a feature list)
- Specific, local, owner-voiced copy (not marketing-speak)
- Irregular visual rhythm (sections feel different from each other)
- Restraint in animation and decoration
- Imperfect, human-feeling compositions

---

### Competitive Landscape Notes

Direct competitors (The Forge, Asgard Games, Halcyon Games — all Houston-area) are:

- Template-driven (Squarespace, Wix, Shopify)
- Retail-store-first in framing (vs. Emporium's venue-first positioning)
- Text-heavy with generic copy
- Weak in photography and visual storytelling

This is a major opportunity. Emporium's site should look like it belongs alongside hospitality venues (bars, restaurants, event spaces), not alongside game stores. The competitive bar is low — but the aspiration should be high.
